Web de escalada con una aplicación de dibujo vectorial para dibujar croquis y geolocalización de zonas y sectores en Google Maps con un servicio REST meteorológico con predicción.
Capacidad de autogestión de la propia bbdd mediante acceso restringido a un Backoffice . Tecnologías utilizadas:
J2EE, JavaScript, JQuery, Ajax, Html5, Css3, MySql, Maven, Servicio Web Rest, JUnit, IDE Eclipse
